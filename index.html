경공급식 사용 가이드

날짜 선택

원하는 날짜의 급식 정보를 확인하는 방법:

  • 상단의 날짜 선택기에서 날짜를 직접 선택
  • 이전/다음 화살표를 클릭하여 날짜 이동
  • 하단의 주간 캘린더에서 날짜 클릭

급식 메뉴 확인

조식, 중식, 석식 메뉴를 확인하는 방법:

  • 상단 탭에서 원하는 급식 시간 선택 (조식, 중식, 석식)
  • 각 메뉴 항목 우측의 아이콘으로 추가 기능 사용 가능

메뉴 상세 정보

각 메뉴 항목 옆의 아이콘 기능:

  • - 메뉴 상세 정보 및 사진 확인
  • - 메뉴 정보 공유하기
  • - 구글에서 메뉴 검색하기

설정 기능

우측 상단의 아이콘을 클릭하여 다양한 설정을 변경할 수 있습니다:

  • 알레르기 정보 강조 표시
  • 폰트 크기 조절
  • 이미지 표시 설정
  • 애니메이션 효과 설정
  • 캐시 데이터 관리

테마 변경

우측 상단의 또는 아이콘을 클릭하여 다크 모드와 라이트 모드를 전환할 수 있습니다.

개인정보 처리방침

개인정보 수집 및 이용

경공급식 서비스는 사용자의 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에 저장되며, 서버로 전송되지 않습니다.

로컬 스토리지 사용

경공급식 서비스는 다음과 같은 목적으로 로컬 스토리지를 사용합니다:

  • 테마 설정 저장
  • 급식 데이터 캐싱 (서버 요청 횟수 감소)
  • 사용자 설정 및 기본 설정 저장

외부 API 사용

경공급식 서비스는 다음과 같은 외부 API를 사용합니다:

  • 나이스 교육정보 개방 포털 API: 급식 정보 조회
  • OpenWeatherMap API: 날씨 정보 조회

외부 API 사용 시 사용자의 개인정보는 포함하지 않습니다.

개인정보 권리

사용자는 언제든지 자신의 기기에 저장된 데이터를 삭제할 수 있습니다. 설정 메뉴의 '캐시 관리'에서 저장된 데이터를 모두 삭제할 수 있습니다.
